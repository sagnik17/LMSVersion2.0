Any Time Library

This is a Yorbit 201 Project on Angular 2.This is self based use website application.

Created By Sagnik Saha (M1037786)

This Web Application Contians the following Features :

  • 1. Self Based Login and Registration Fucntionality
  • 2. User can self issue and return the book.User can issue a maximum of 5 books at a time.User can also Edit their Profile Information.
  • 3. User can also filter the books upon category.They can also view the book details with their cover photo.
  • 4. Admin can Add Books, Edit Books Details, Delete Books and can also view books information.Admin can also view the list of books issued to different Users.
